|жөҸи§ҲйҮҸ:

й—®йўҳжҸҸиҝ°

еңЁејҖеҸ‘дёҖдёӘи®ўеҚ•зі»з»ҹж—¶пјҢйҒҮеҲ°дәҶдёҖдёӘеҘҮжҖӘзҡ„й—®йўҳпјҡ

  1. з”ЁжҲ·дёӢеҚ•еҗҺпјҢеә“еӯҳжүЈеҮҸжҲҗеҠҹ
  2. дҪҶи®ўеҚ•зҠ¶жҖҒжІЎжңүжӣҙж–°дёәвҖқе·Іж”Ҝд»ҳвҖқ
  3. жҹҘзңӢж—Ҙеҝ—пјҢжІЎжңүеҸ‘зҺ°д»»дҪ•ејӮеёё

д»Јз ҒеңәжҷҜ

й—®йўҳеҲҶжһҗ

1. дәӢеҠЎйҡ”зҰ»зә§еҲ«

жҹҘзңӢж•°жҚ®еә“й…ҚзҪ®пјҢеҸ‘зҺ°дҪҝз”Ёзҡ„жҳҜREAD COMMITTEDйҡ”зҰ»зә§еҲ«гҖӮ

2. е№¶еҸ‘еңәжҷҜ

еҒҮи®ҫжңүдёӨдёӘиҜ·жұӮеҗҢж—¶еӨ„зҗҶпјҡ

ж—¶й—ҙзәҝпјҡ

  1. T1: дәӢеҠЎAејҖе§ӢпјҢжҹҘиҜўеә“еӯҳдёә100
  2. T2: дәӢеҠЎBејҖе§ӢпјҢжҹҘиҜўеә“еӯҳдёә100
  3. T3: дәӢеҠЎAжүЈеҮҸ10пјҢеә“еӯҳеҸҳдёә90пјҢдҪҶжңӘжҸҗдәӨ
  4. T4: дәӢеҠЎBжүЈеҮҸ10пјҢеә“еӯҳеҸҳдёә90пјҲеҹәдәҺ100и®Ўз®—пјүпјҢдҪҶжңӘжҸҗдәӨ
  5. T5: дәӢеҠЎAжҸҗдәӨпјҢеә“еӯҳдёә90
  6. T6: дәӢеҠЎBжҸҗдәӨпјҢеә“еӯҳдёә90пјҲеә”иҜҘжҳҜ80пјҒпјү

й—®йўҳпјҡеҮәзҺ°дәҶи¶…еҚ–пјҒ

3. йҡ”зҰ»зә§еҲ«еҜ№жҜ”

йҡ”зҰ»зә§еҲ« и„ҸиҜ» дёҚеҸҜйҮҚеӨҚиҜ» е№»иҜ» иҜҙжҳҺ
READ UNCOMMITTED еҸҜиғҪ еҸҜиғҪ еҸҜиғҪ жңҖдҪҺзә§еҲ«
READ COMMITTED дёҚеҸҜиғҪ еҸҜиғҪ еҸҜиғҪ MySQLй»ҳи®Ө
REPEATABLE READ дёҚеҸҜиғҪ дёҚеҸҜиғҪ еҸҜиғҪ еҸҜйҮҚеӨҚиҜ»
SERIALIZABLE дёҚеҸҜиғҪ дёҚеҸҜиғҪ дёҚеҸҜиғҪ жңҖй«ҳзә§еҲ«пјҢжҖ§иғҪжңҖе·®

и§ЈеҶіж–№жЎҲ

ж–№жЎҲдёҖпјҡдҪҝз”ЁREPEATABLE READпјҲдёҚжҺЁиҚҗпјү

й—®йўҳпјҡиҷҪз„¶иғҪи§ЈеҶідёҚеҸҜйҮҚеӨҚиҜ»пјҢдҪҶжҖ§иғҪиҫғе·®пјҢдё”еҸҜиғҪеҮәзҺ°жӯ»й”ҒгҖӮ

ж–№жЎҲдәҢпјҡдҪҝз”ЁжӮІи§Ӯй”ҒпјҲSELECT FOR UPDATEпјү

SQLпјҡ

дјҳзӮ№пјҡз®ҖеҚ•зӣҙжҺҘпјҢдҝқиҜҒж•°жҚ®дёҖиҮҙжҖ§
зјәзӮ№пјҡжҖ§иғҪиҫғе·®пјҢеҸҜиғҪйҖ жҲҗй”Ғзӯүеҫ…

ж–№жЎҲдёүпјҡдҪҝз”Ёд№җи§Ӯй”ҒпјҲжҺЁиҚҗпјү

SQLпјҡ

дјҳзӮ№пјҡжҖ§иғҪеҘҪпјҢйҖӮеҗҲй«ҳе№¶еҸ‘еңәжҷҜ
зјәзӮ№пјҡйңҖиҰҒйҮҚиҜ•жңәеҲ¶пјҢе®һзҺ°зЁҚеӨҚжқӮ

ж–№жЎҲеӣӣпјҡдҪҝз”ЁеҲҶеёғејҸй”Ғ

жңҖз»Ҳж–№жЎҲ

з»“еҗҲдёҡеҠЎеңәжҷҜпјҢжҲ‘д»¬йҮҮз”ЁдәҶд№җи§Ӯй”Ғ + еҲҶеёғејҸй”Ғзҡ„з»„еҗҲж–№жЎҲпјҡ

  1. еҲҶеёғејҸй”ҒпјҡйҳІжӯўе№¶еҸ‘иҜ·жұӮеҗҢж—¶иҝӣе…Ҙ
  2. д№җи§Ӯй”Ғпјҡж•°жҚ®еә“еұӮйқўдҝқиҜҒж•°жҚ®дёҖиҮҙжҖ§

з»ҸйӘҢжҖ»з»“

  1. зҗҶи§Јйҡ”зҰ»зә§еҲ«пјҡдёҚеҗҢйҡ”зҰ»зә§еҲ«и§ЈеҶізҡ„й—®йўҳдёҚеҗҢпјҢиҰҒж №жҚ®дёҡеҠЎйҖүжӢ©
  2. е№¶еҸ‘жҺ§еҲ¶пјҡй«ҳе№¶еҸ‘еңәжҷҜдёӢпјҢиҰҒиҖғиҷ‘е№¶еҸ‘жҺ§еҲ¶жңәеҲ¶
  3. й”Ғзҡ„йҖүжӢ©пјҡ
    • жӮІи§Ӯй”ҒпјҡйҖӮеҗҲеҶҷеӨҡиҜ»е°‘зҡ„еңәжҷҜ
    • д№җи§Ӯй”ҒпјҡйҖӮеҗҲиҜ»еӨҡеҶҷе°‘зҡ„еңәжҷҜ
    • еҲҶеёғејҸй”ҒпјҡйҖӮеҗҲеҲҶеёғејҸзҺҜеўғ
  4. жҖ§иғҪе№іиЎЎпјҡеңЁж•°жҚ®дёҖиҮҙжҖ§е’ҢжҖ§иғҪд№Ӣй—ҙжүҫеҲ°е№іиЎЎ

ж•ҷи®ӯпјҡдәӢеҠЎйҡ”зҰ»зә§еҲ«дёҚжҳҜдёҮиғҪзҡ„пјҢй«ҳе№¶еҸ‘еңәжҷҜдёӢйңҖиҰҒз»“еҗҲй”ҒжңәеҲ¶жқҘдҝқиҜҒж•°жҚ®дёҖиҮҙжҖ§гҖӮзҗҶи§Јеҗ„з§Қй”Ғзҡ„йҖӮз”ЁеңәжҷҜеҫҲйҮҚиҰҒгҖӮ
